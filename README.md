# Проєкт Різдво

Це веб-застосунок Django, який показує, чи сьогодні Різдво, і містить гру "Таємний Санта".

## Функції

- Показує, чи сьогодні Різдво.
- Гра "Таємний Санта" з управлінням учасниками.

## Вимоги

- Python 3.8+
- Django 5.1+
